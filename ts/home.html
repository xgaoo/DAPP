Overview

De Novo AML patient Prognostic Prediction (DAPP) is developed for biologists and clinical researchers to predict the prognosis of de novo Acute Myeloid Leukemia (AML) using machine learning models. DAPP supports both gene mutation and expression data as input.

We searched published AML data sets with survival information. Only keep non-redundant de novo AML samples with survival information, a total of 3,315 samples were retained, including 1,668 gene expression samples and 1,647 mutation samples. Then, we employed nine machine learning methods to build multiple models. Final, we adjust the scores with a smooth monotonic fit to keep the survival probability score decreases monotonously with time after the diagnosis.

Gene Expression Datasets

Gene Expression Datasets Description

Microarray data

GSE37642

The series matrix files of GSE37642 were downloaded from NCBI Gene Expression Omnibus (GEO). 9 samples with missing survival information were removed, and 553 cases were retained.

GSE6891

The raw CELL files for GSE6891 were downloaded from NCBI Gene Expression Omnibus (GEO). The overall survival data was downloaded from cytopred's github repository https://github.com/narenschandran/cytopred (Salunkhe, et al., 2018). Samples with missing age or survival information were removed, and 505 cases were retained.

RNA-seq data

GSE106291

The raw reads count files for GSE106291 were downloaded from NCBI Gene Expression Omnibus (GEO). Then calculate the RPKM values of all genes in all samples using the R/Bioconductor package edgeR version 3.24.3.

TCGA & BEAT

The mRNA expression Z-Scores file of BEAT AML dataset and the mRNA gene expression file of TCGA AML dataset were downloaded from cbioportal (http://www.cbioportal.org/).

Only the de novo AML patients with overall survival information were kept for this study. GSE106291(n=250), BEAT(n=187) and TCGA(n=173) were retained.

Mutation Datasets

Mutation Datasets Description

The three datasets were used for the Mutation models.

The data of AMLSG was downloaded from AML-multistage's github repository http://www.github.com/gerstung-lab/aml-multistage (Gerstung, et al., 2017). The mutations data of BEAT AML and TCGA AML dataset were downloaded from cbioportal (http://www.cbioportal.org/). Only the de novo AML patients with overall survival information were kept for this study. AMLSG(n=1234), BEAT(n=217) and TCGA(n=196) were retained.
