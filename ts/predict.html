File format descriptions

The file consists of two columns, separated by commas:

1. First column is feature name (clinical / expression / mutation):
   a. clinical features with prefix 'cli_'
   b. gene expression features with 'exp_'
   c. mutaiton features(include karyotype) with 'mut_'

2. Second column is values for features:
   mutation features: value is 0/1, represent wild/mutant type
   expression features: value is log2 transfermed RPKM value

The file name must end with .csv/.csv.gz
